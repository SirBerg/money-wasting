# Tracking your time wasted in meetings
Did you ever wonder how much money you could be saving if you just skipped all those pointless meetings?
Well now you can find out!
This simple tool allows you to log your meetings and calculate the total time (and money) spent in them.
Go to [https://money.boerg.co/](https://money.boerg.co/) to get started!
Everything is stored on your device, so your data is safe and private (even the leaderboard).
## Development
To set up the development environment, follow these steps:
1. Set up bun
2. Install dependencies:
   ```bash
   bun install
   ```
3. Start the development server:
```bash
   bun dev
   ```
4. Open your browser and navigate to `http://localhost:4321` to see the application in action.
